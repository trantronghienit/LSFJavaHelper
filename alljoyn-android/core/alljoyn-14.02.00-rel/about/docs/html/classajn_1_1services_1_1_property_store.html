Member Enumeration Documentation

enum ajn::services::PropertyStore::Filter

Filter has three possible values ANNOUNCE, READ,WRITE READ is for data that is marked as read ANNOUNCE is for data that is marked as announce WRITE is for data that is marked as write.

Enumerator:
ANNOUNCE 

ANNOUNCE Property that has ANNOUNCE enabled.

READ 

READ Property that has READ enabled.

WRITE 

WRITE Property that has WRITE enabled.
