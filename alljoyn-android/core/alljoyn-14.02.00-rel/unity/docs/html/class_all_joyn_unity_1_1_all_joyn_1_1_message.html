out object  value 
) [inline]

Unpack and return the arguments for this message.

This method uses the functionality from AllJoyn.MsgArg.Get(string, out object) see MsgArg.cs for documentation.

Parameters:
signatureThe signature to match against the message arguments.
valueobject unpacked values the values into
Returns:
  • QStatus.OK if the signature matched and MsgArg was successfully unpacked.
  • QStatus.BUS_SIGNATURE_MISMATCH if the signature did not match.
  • An error status otherwise

uint AllJoynUnity.AllJoyn.Message.TimeStamp [get]

Returns the timestamp (in milliseconds) for this message.

If the message header contained a timestamp this is the estimated timestamp for when the message was sent by the remote device, otherwise it is the timestamp for when the message was unmarshaled. Note that the timestamp is always relative to local time.

Returns:
The timestamp for this message.
