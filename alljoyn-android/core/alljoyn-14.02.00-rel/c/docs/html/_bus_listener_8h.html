Detailed Description

BusListener is an abstract base class (interface) implemented by users of the AllJoyn API in order to asynchronously receive bus related event information.

Type for the FoundAdvertisedName callback.

Called by the bus when an external bus is discovered that is advertising a well-known name that this attachment has registered interest in via a DBus call to org.alljoyn.Bus.FindAdvertisedName

Parameters:
contextThe context pointer that was passed into the alljoyn_buslistener_create function

typedef void(* alljoyn_buslistener_lost_advertised_name_ptr)(const void *context, const char *name, alljoyn_transportmask transport, const char *namePrefix)

Type for the LostAdvertisedName callback.

Called by the bus when an advertisement previously reported through FoundName has become unavailable.

Parameters:
