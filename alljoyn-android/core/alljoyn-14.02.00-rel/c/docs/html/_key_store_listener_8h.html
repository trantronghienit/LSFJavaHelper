Detailed Description

The alljoyn_keystorelistener handles requests to load or store the key store.

Typedef Documentation

typedef struct _alljoyn_keystorelistener_handle* alljoyn_keystorelistener

An application can provide a key store listener to override the default key store load request and store request behavior.

If an alljoyn_keystorelistener is provided it will override the default key store behavior.

typedef QStatus(* alljoyn_keystorelistener_loadrequest_ptr)(const void *context, alljoyn_keystorelistener listener, alljoyn_keystore keyStore)

Type for the LoadRequest callback.

This function is called when a key store needs to be loaded.

Remarks:
The application must call alljoyn_keystorelistener_putkeys to put the new key store data into the internal key store.
Parameters:
contextThe context pointer passed into the alljoyn_keystorelistener_create function
listenerThe listener that is responsible for the LoadRequest callback. This listener should be used when calling the alljoyn_keystorelistener_putkeys function.
keyStoreReference to the alljoyn_keystore to be loaded.
Returns:
  • ER_OK if the load request was satisfied
  • An error status otherwise
typedef QStatus(* alljoyn_keystorelistener_storerequest_ptr)(const void *context, alljoyn_keystorelistener listener, alljoyn_keystore keyStore)

Type for the StoreRequest callback.

This function is called when a key store needs to be stored.

Remarks:
The application must call alljoyn_keystorelistener_getkeys to obtain the key data to be stored.
Parameters:
contextThe context pointer passed into the alljoyn_keystorelistener_create function
listenerThe listener that responsible for the StoreRequest callback. This listener should be used when calling the alljoyn_keystorelistener_getkeys function.
keyStoreReference to the alljoyn_keystore to be stored.
Returns:
  • ER_OK if the store request was satisfied
  • An error status otherwise

alljoyn_keystore  keyStore,

Get the current keys from the key store as an encrypted byte string. 

 QStatus status;
 char* sink;
 size_t sink_sz;
 alljoyn_keystorelistener_getkeys(listener, keyStore, NULL, &sink_sz);
 sink = (char*) malloc(sizeof(char) * sink_sz);
 status = alljoyn_keystorelistener_getkeys(listener, keyStore, sink, &sink_sz);
 assert(status == ER_OK);
Parameters:
listenerThe alljoyn_keystorelistener from which to get the keys.
keyStoreThe keyStore to get from. This is the keystore indicated in the StoreRequest call.
sinkThe byte string to write the keys to. If sink is NULL then the sink_sz parameter will return the size of the sink plus the terminating nul character.
[in,out]sink_szThe size of the byte string provided. This will return the actual size of the sink plus the terminating nul character. If the sink_sz out should be larger than or equal to the sink_sz in.
Returns:
  • ER_OK if successful
  • ER_BUFFER_TOO_SMALL if the specified sink_sz is is not large enough to hold the sink
  • An error status otherwise
