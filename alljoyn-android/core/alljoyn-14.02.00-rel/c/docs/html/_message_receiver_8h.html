Detailed Description

MessageReceiver is a base class implemented by any class which wishes to receive AllJoyn messages.

typedef void(* alljoyn_messagereceiver_methodhandler_ptr)(alljoyn_busobject bus, const alljoyn_interfacedescription_member *member, alljoyn_message message)

MethodHandlers are MessageReceiver functions which are called by AllJoyn library to forward AllJoyn method_calls to AllJoyn library users.

Parameters:
busThe bus object triggering this callback.
memberMethod interface member entry.
messageThe received method call message.
typedef void(* alljoyn_messagereceiver_replyhandler_ptr)(alljoyn_message message, void *context)

ReplyHandlers are MessageReceiver functions which are called by AllJoyn library to forward AllJoyn method_reply and error responses to AllJoyn library users.

Parameters:
messageThe received message.
contextUser-defined context passed to MethodCall and returned upon reply.
