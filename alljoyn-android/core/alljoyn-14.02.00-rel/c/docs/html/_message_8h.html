Returns a string that provides a brief description of the message.

Parameters:

[out]strThe character string that will hold the description of the message

)

If the message is an error message returns the error name and optionally the error message string.

Parameters:

[out]errorMessage
  • Return the error message string stored
  • leave errorMessage unchanged if error message string not found
[in,out]errorMessage_sizesize of the errorMessage string (including the terminating nul byte) if errorMessage == NULL this will be changed to the size of the string so a buffer can be allocated to hold the string.
Returns:
  • If error detected return error name stored in the AllJoyn header field
  • NULL if error not detected

AJ_API uint32_t alljoyn_message_gettimestamp ( alljoyn_message  msg)

Returns the timestamp (in milliseconds) for this message.

If the message header contained a timestamp this is the estimated timestamp for when the message was sent by the remote device, otherwise it is the timestamp for when the message was unmarshaled. Note that the timestamp is always relative to local time.

Parameters:
[in]msgThe alljoyn_message from which to extract the timestamp information.
Returns:
The timestamp for this message.
