AllJoyn session related data types.

Typedef Documentation

typedef uint16_t alljoyn_sessionport

alljoyn_sessionport identifies a per-alljoyn_busattachment receiver for incoming JoinSession requests.

alljoyn_sessionport values are bound to an alljoyn_busattachment when the attachment calls alljoyn_busattachment_bindsessionport.

NOTE: Valid alljoyn_sessionport values range from 1 to 0xFFFF.

AJ_API QCC_BOOL alljoyn_sessionopts_iscompatible ( const alljoyn_sessionopts  one,
const alljoyn_sessionopts  other 
)

Determine whether one alljoyn_sessionopts is compatible with the alljoyn_sessionopts offered by other.

Compatibility means that the alljoyn_sessionopts share at least one of each

  • transport type
  • traffic type
  • proximity type

Note that multipoint support is not a condition of compatibility

Parameters:
oneOptions to be compared against other.
otherOptions to be compared against one.
Returns:
QCC_TRUE iff this alljoyn_sessionopts can use the option set offered by other.
